Computer Science Projects

Capstone Project: Playpaws

Overview: This project, Playpaws, is a mobile application concept developed in a team of four. The goal was to create a platform for dog owners to easily arrange playdates for their pets, fostering socialization and community. While a live demo video isn't available here, this description highlights the key functionalities and my specific contributions to the application.

Goal: Playpaws aimed to solve the challenge of connecting dog owners with compatible dogs in their local area for playdates, thereby promoting dog well-being and owner interaction.

Technologies Demonstrated (Conceptual): Flutter/Dart (for the calendar feature), Firebase/Firestore (for backend data storage), Machine Learning (for dog compatibility matching), Embedding Models (for feature-based matching).

My Role: As part of the development team, I was primarily responsible for designing, developing, and integrating the calendar feature using Flutter and Dart. This included ensuring seamless data storage and retrieval using Firestore. I also contributed to the conceptualization and implementation strategy for the dog compatibility matching system utilizing machine learning and embeddings.

Key Features: Intuitive Calendar Feature:The application includes a user-friendly calendar interface (developed by me) for scheduling and managing dog playdates. Intelligent Dog Matching: Playpaws utilizes a combination of machine learning and embedding techniques to suggest compatible playmates based on a calculated score. User Profiles & Dog Profiles: The app allows users to create detailed profiles for themselves and their dogs, providing necessary information for matching.
